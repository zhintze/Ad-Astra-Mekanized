## Biome Modifier Template

```json
{
  "neoforge:conditions": [
    {
      "type": "neoforge:mod_loaded",
      "modid": "MOD_ID_HERE"
    }
  ],
  "type": "neoforge:add_spawns",
  "biomes": "#adastramekanized:PLANET_biomes",
  "spawners": [
    {
      "type": "namespace:entity_id",
      "weight": 10,
      "minCount": 1,
      "maxCount": 2
    }
  ]
}
```

### Weight Guidelines
- **Common mobs:** weight 30-50
- **Uncommon mobs:** weight 10-25
- **Rare mobs:** weight 3-8
- **Very rare/bosses:** weight 1-2
